Change Detection Demo

Each time you click a button, Angular decides whether to rerender the component based on its change detection strategy:

  • ✅ Default Strategy: Always updates the UI when any property changes.
  • 🚀 OnPush Strategy: Only updates when a new object reference is assigned or an event inside the component triggers detection.

🔹 If the UI doesn’t update in OnPush mode, it's because the object reference didn’t change, even though its values did.

🔹 To force UI updates in OnPush mode, assign a new object/array/map reference.

💡 Watch the counter to see how often each component rerenders!
